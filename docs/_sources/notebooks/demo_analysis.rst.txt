demo_analysis
========

This section showcases example use cases for MissMecha.
Each notebook demonstrates a different missingness generation or evaluation scenario.





Generate Missing Data
======================

This example demonstrates how to simulate MCAR, MAR, and MNAR mechanisms using the `MissMechaGenerator`.

You will see:
- How to specify different missing mechanisms
- What the boolean masks look like
- How to visually inspect generated patterns

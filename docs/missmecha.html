Compute and present detailed missingness statistics for each column, along with overall missing rate. Supports both DataFrame and ndarray.

Parameters:

  • data (pd.DataFrame or np.ndarray) – Input dataset.

  • print_summary (bool, default=True) – Whether to print overall missingness summary.

  • plot (bool, default=False) – Whether to show a barplot of missing rate per column.

Returns:

result –

{

“report”: pd.DataFrame of column-wise missing info, “overall_missing_rate”: float (total % of missing cells)

}

dict

missmecha.analysis.evaluate_imputation(ground_truth: DataFrame, filled_df: DataFrame, incomplete_df: DataFrame, method: str, status: dict | None = None)[source]¶

Evaluate imputation performance by comparing filled values to ground truth at missing positions.

Parameters:

  • ground_truth (pd.DataFrame) – Fully observed reference dataset.

  • filled_df (pd.DataFrame) – The dataset after imputation.

  • incomplete_df (pd.DataFrame) – Dataset with original missing values (to locate evaluation positions).

  • method (str) – ‘rmse’, ‘mae’, or ‘avgerr’ — determines how numerical columns are evaluated.

  • status (dict, optional) – A dictionary mapping column names to variable types: ‘num’ for numerical, ‘cat’ or ‘disc’ for categorical. If not provided, all variables are treated as numerical.

Returns:

result –

{

“column_scores”: {col_name: score, …}, “overall_score”: average_score

}

Return type:

dict

Visual Module
=============

This module provides visualization tools for inspecting missing data patterns.  
It includes matrix-style plots, nullity correlation heatmaps, and column-level summaries.  
These tools are especially helpful during exploratory data analysis (EDA) and when diagnosing the missingness structure.
